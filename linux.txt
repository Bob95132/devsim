This is the DEVSIM distribution.  Extract the files from
the archive into your preferred directory.  Product
documentation is available in the devsim/doc directory.

DEVSIM depends on the following 64 bit applications installed on your system:

* Python 2.7
* Tcl 8.6
* Intel MKL

All of these dependencies are available by installing:
Anaconda Python 2.7
https://continuum.io/download

Paths may be set by editing and then sourcing the appropriate script for your shell.

csh:
  source anaconda_vars.csh

bash:
  source anaconda_vars.sh

If you have problems running the software, please file an issue at:
https://github.com/devsim/devsim
